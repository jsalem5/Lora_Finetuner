To download either do:
    pip install git+https://github.com/jsalem5/Lora_Finetuner@92bcf798afdd35f99f2ed149d74d250c6e57ffae
or do:
    1. First download dependencies:
        pip install -r requirement.txt
    2. then do "pip install ." inside of the project folder
    3. The cli name is "lora-finetuner"
    4. Use "lora-finetuner --help" to see user manual
